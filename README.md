# tlias-web-management

仓库存储了黑马程序员 Java Web (2024) 课程项目的后端部分, 主要包含以下内容:

- 基础 CRUD
  - RESTful
  - 三层架构
- MySQL
  - Mybatis
  - 多表操作
  - Spring 事务管理
- 日志技术
  - Logback
- 异常处理
  - Spring 全局异常处理器
  - 自定义异常
- 文件上传
  - 阿里云 OSS
- 登陆校验
  - JWT 令牌
  - Spring 拦截器
- "增删改"日志统一管理
  - Spring AOP
  - ThreadLocal
